## 盒型碰撞体

盒型碰撞体是一种基本的长方形原始碰撞体

### 属性

| 属性：     | 功能：                                                       |
| ---------- | ------------------------------------------------------------ |
| Is Trigger | 如果启用此属性，则该碰撞体将作用于触发事件，并被物理引擎忽略。 |
| Material   | 引用物理材质，可确定该碰撞体与其他对象的交互方式。           |
| Center     | 碰撞体在对象局部空间中的位置                                 |
| Size       | 碰撞体在X、Y、Z方向上的大小                                  |

盒型碰撞体是可用于板条箱的长方体。但是，可以使用薄形盒作为地板、墙壁或坡道。盒型碰撞体也是复合碰撞体中的有用元素。
